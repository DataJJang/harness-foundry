# Agent Handoff Checklist

- [ ] handoff 발신 역할과 수신 역할이 명시되었는가
- [ ] 작업 목적과 범위가 한 문장으로 정리되었는가
- [ ] 관련 파일 경로가 포함되었는가
- [ ] 관련 문서 경로가 포함되었는가
- [ ] `.agent-base/agent-workboard.json`의 현재 lane과 owned path가 반영되었는가
- [ ] 이미 끝난 결정 사항이 정리되었는가
- [ ] 미해결 이슈가 정리되었는가
- [ ] 필요한 검증 상태가 기록되었는가
- [ ] DB/보안/운영 영향이 있는지 표시되었는가
- [ ] 수신 역할이 무엇을 출력해야 하는지 명시되었는가
- [ ] 다음 handoff 대상이 필요한지 표시되었는가
- [ ] `docs/ai/agent-handoff-log.md`에 baton 기록이 남았는가
