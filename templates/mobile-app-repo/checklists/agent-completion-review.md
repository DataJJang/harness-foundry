# Agent Completion Review Checklist

- [ ] 역할별 책임 범위가 완료되었는가
- [ ] 요청 범위 밖 변경이 없는가
- [ ] 필요한 검증이 수행되었는가
- [ ] 미검증 항목이 기록되었는가
- [ ] 문서 영향이 반영되었는가
- [ ] `.agent-base/agent-workboard.json`의 해당 lane이 최신 상태인가
- [ ] DB 영향이 있으면 data-steward 산출물이 있는가
- [ ] 보안 영향이 있으면 security-review 결과가 있는가
- [ ] handoff 누락이 없는가
- [ ] release risk 또는 운영 영향이 기록되었는가
- [ ] orchestrator 또는 최종 담당자가 완료를 선언할 근거가 준비되었는가
