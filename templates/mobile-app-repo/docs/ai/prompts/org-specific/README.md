# Optional Organization-Specific Prompt Packs

이 디렉토리는 공통 prompt library 위에 얹는 조직 또는 제품군 전용 prompt pack을 담는다.

기본 배포본은 공통 prompt만으로도 프로젝트 생성과 문서 작업을 수행할 수 있어야 한다.  
조직 전용 prompt는 아래 조건일 때만 추가로 사용한다.

- 저장소 구조나 네이밍이 조직 표준에 강하게 묶여 있다
- 고정된 운영 문서 형식이나 검증 형식이 있다
- 특정 제품군이나 플랫폼에만 통용되는 경로/용어/배포 절차가 있다

현재 포함된 예시:

- [`aitelecom/README.md`](./aitelecom/README.md)
- [`egov-public-sector/README.md`](./egov-public-sector/README.md)
